# 오징어게임 심리테스트 

## 🦑 전세계는 지금 오징어게임 열풍!

넷플릭스 드라마 "오징어 게임"에 과몰입한 사람들은 주목!!

실제 드라마의 장면들로 구성된 심리테스트를 진행하면서 
내가 어떤 캐릭터와 비슷한지 분석해봐요 ! ㅇㅅㅁ
